Mediacreeper

Visualize recent visits from media companies with MediaCreeper

Description

MediaCreeper tracks visitors from media companies using the API available from MediaCreeper.com. Visits are visualized inside the admin area with a graph in the Dashboard Widget. Each post gets its own metabox in the editor with information about recent visits by media companies to that post.

See MediaCreeper.com and MediaCreeper.com API for more details.

Download

Download mediacreeper.1.0.zip now.

Installation

  1. Upload the mediacreeper directory to the /wp-content/plugins/ directory
  2. Activate the plugin through the 'Plugins' menu in WordPress
